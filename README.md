# 🌿 GreenDen – Plant Shopping Website

GreenDen is a modern and responsive plant shopping website built using **HTML5** and **Tailwind CSS**. This project was developed to practice utility-first CSS, responsive web design, and creating clean, user-friendly interfaces.

## 🚀 Live Demo

🌐 **Website:**  
https://priyadharshiniselvarajj.github.io/greenden-tailwind/

## 📌 Features

- 🌿 Modern and responsive user interface
- 🏡 Attractive hero section
- 🌱 Featured plant collection
- 💚 About Us section
- ⭐ Customer reviews section
- 📞 Contact page
- 📱 Fully responsive design for desktop, tablet, and mobile devices
